HTML Semantics CheetSheet

Semantics Tags

Tag Name Description
< header > Webpage Header The Header helps seperate the code for the top of the page
< nav > Navigation The Nav seperates all code for navigation (home, about us, etc.)
< main > Main section The Main section includes most of the code for the webpage.
< section > Section block Seperates a certain part of the page, like a chapter in a book.
< article > Article section The Article specifies independent, self-contained content.
< footer > Footer Section The Footer includes everyting at the bottom (contact details, copyright, etc.)
